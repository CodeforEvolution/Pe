On this page you can control the way softwrapping initially behaves. You can turn it on or off by checking SoftWrap.

If you choose to wrap files, you have three possibilities. The first is that you can wrap files to the width of a window. Whenever you resize a window in that case, all of the contents will be rewrapped, which may take a while.

The second possibility is to wrap to the width of a piece of paper. But since Pe cannot print, this is only interesting for wrapping the contents to the width of some arbitrary amount. Currently, this is set to 300 pixels. When printing is available, this width will be the real width of a piece of paper.

The third and last possibility is to wrap lines to a predefined number of characters. This is useful for text that will be displayed on terminals e.g. Please note that I said characters and not bytes; in UTF-8 a line may consist of more bytes than characters.
