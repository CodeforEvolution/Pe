Pe Manual

Introduction

This is the manual for Pe, the programmers and HTML editor for BeOS. This document assumes you are already familiar with the basic concepts of BeOS, if you are not, I recommend that you read the documentation for BeOS first.

This manual consists of two parts. The first is a guided tour around the interface of Pe. The second part is more like a reference guide, where you can find information on how to execute several tasks.
