Language Support

Pe has syntax colouring, a function popup menu and balancing. All three of these functions depend on the language in which the text is written. To make it easy to extend the number of supported languages, Pe introduced the concept of language plug-ins. These plug-ins provide the three functions mentioned above.

The language plug-ins should be put into the directory called Languages inside the directory containing the Pe application.

Pe comes with eight language plug-ins, they are C/C++, HTML, Oberon-2, Java, TeX, Rez, Perl and Python. If you have the need for any other language, please let me know and supply me with a description of it, and I will see if it can be added.
You can also create a plug-in yourself. When you've installed the Goodies package with the Pe Installer, you will have the necessary interface file containing detailed descriptions on how to build your own plug-in.
