Error and Search Result Window

This window is used to report errors and search results to the user. The elements of this list often contain references to pieces of text in documents.
If you double click an item, Pe will open the document and select the identified part of text.

It is currently not possible to save the contents of such a window. If you think this should be a feature, please let me know and I might add it in a future release.
