State

State information is information stored in the attributes of a file, recording things like window position, font family and size. You can choose either to restore these settings when a file is opened or to ignore the information.

The first four options speak for themselves. The fifth has to do with worksheets and documents containing shell commands. If you choose to restore the working directory, you won't have to select a working directory first when you want to execute these commands after opening a file containing them.

The Use State Format popup lets you choose the format in which the state information will be stored. Pe has its own format which is different from the one MetroWerks BeIDE uses. Here you can choose which one to use by default. If you choose BeIDE, files, when opened in BeIDE, will have the same font and position as they had when you saved them in Pe.
