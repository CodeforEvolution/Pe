BeIDE Interface

When you've checked the checkbox on the Connections page of the preferences dialog, every window will have a BeIDE menu. This menu allows you to do several simple tasks involving Metrowerks BeIDE.

Unfortunately the current version of BeIDE doesn't support external editors yet. Therefore it's not possible to fully integrate Pe with BeIDE. But as soon as BeIDE does support external editors, Pe will be among the first to exploit this.

I have heard from reliable sources that Be is working on adding support for external editors, so hang on.
