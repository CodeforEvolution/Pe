Connections

If you check the first checkbox on this page, Pe will show a BeIDE menu in the menubar for every document. This menu allows you to connect to BeIDE.

The second checkbox will let Pe load the extensions found in the Metrowerks BeIDE editor add-ons folder. Since many extensions written for BeIDE implement functionality already available in Pe, this has become an option.
