Groups

If you check the Sort... checkbox, the files inside a group will always be sorted. Otherwise the files will end up in the order they were added to the group. Sorry, there is no way yet to organize a group any better, but it is on my list.

Relative paths will be stored inside a group file if you check the second checkbox. This means that when you move a groupfile to another location (e.g. by renaming it's enclosing directory) you won't lose the references to other files in this directory.
