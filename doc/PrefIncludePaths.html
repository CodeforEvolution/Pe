Include Paths

This pane can be used to influence Pe's search method for included files. Pe looks for include files when you choose one such file from the function popup or when you choose Open Selection from the File Menu.

If you check Parent, Pe will start looking for the include file in the parent directory of the file currently being edited, if any.

If you check the option with $BEINCLUDES, then Pe will continue its search for the file in all the paths stored in this environmental variable. This variable is also used by the compilers to check the location of the standard include files.

And, above all, you can add your own paths. Just drag the directories containing your library headers onto the path box and it will be added. You can also alter paths already in this box or delete them.
