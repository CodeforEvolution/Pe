File Types

Use this pane to change the contents of the various File Type (sometimes called Mime Type) Popup Menus. The mimetype of a file determines what the Tracker will do when you double click it. If it is set to text/html e.g. NetPositive will be launched by default.

You can add your own types by typing it in the edit box and then clicking Add. Add will be dimmed if the type is not valid. You can also change or delete an item from the list. To add a new item, you can also drag a file with that kind of mimetype onto this box.
