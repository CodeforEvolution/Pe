Startup

In this pane you can select what to do when Pe is launched. Since every BeOS application needs at least one window in order to stay alive, well, to be useful anyway, the option Do Nothing only operates if you have also selected the option Always keep a Worksheet Open from the Worksheet pane. In that case, when Pe is launched only the Worksheet will be opened and no new document, which is the default behaviour for an application like this.

Show Open Dialog opens the standard Open File Dialog so you can choose the file which you want to open. Create New Document will, as you will have guessed, create a new document when Pe is launched.

You can also choose to have the HTML Palette show up everytime you lauch Pe.
